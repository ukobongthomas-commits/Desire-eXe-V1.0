### **Prerequisites**
- Node.js 20+ 
- WhatsApp Account
- Gemini API Key (for AI features)
- OpenRouter API Key

### **Local Installation**
```bash

git clone https://github.com/Desire-exe/Desire-eXe-V1.0
cd Desire-eXe-V1.0 

# Install dependencies
npm install --legacy-peer-deps

# Start the bot
```
